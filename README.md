# Sales and Customer Pattern Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on sales and customer data to uncover purchasing behavior, sales trends, and business insights.

## Objectives
- Analyze customer purchasing patterns
- Identify sales trends
- Explore revenue distribution
- Visualize key business metrics

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis
- Data Cleaning
- Missing Value Analysis
- Customer Behavior Analysis
- Sales Trend Analysis
- Data Visualization
